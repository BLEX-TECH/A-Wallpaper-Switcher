# A-Wallpaper-Switcher
A high-performance, blurred-grid wallpaper selector for Hyprland. Designed for speed and aesthetics, providing a 12-item visual gallery with smooth transitions.

Made by **BlexTech**.

## ✨ Features
- **Blur Gallery:** Transparent Rofi window with Hyprland blur support.
- **12-Item Grid:** 4x3 layout for easy navigation.
- **Smart Detection:** Automatically uses `swww` or `awww` based on what is installed.
- **Random Mode:** Set a random wallpaper with a single command.
- **ZSH Powered:** Faster list generation and better file globbing.
